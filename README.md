# Парсер диалогов ExM в JSON

Данный скрипт поможет вам извлечь диалоги из файлов `strings.xml` или `dialogsglobal.xml` в файл формата `.json`.

В случае выбора `dialogsglobal.xml` файла структурирует диалог, чтобы он выглядел целостно.

Так-же скрипт подсчитывает количество реплик, слов и символов у конкретного персонажа.

---

## Запуск из исходного кода:

1. Установить Python версии минимум [3.12.6](https://www.python.org/downloads/release/python-3126/), если он ещё не установлен.

2. Открыть командную строку, перейти в папку, где находится файл и установить зависимости `(easygui, lxml)`:
```sh
pip install -r requirements.txt
```

3. Запустить скрипт:
```sh
python ./DialogParse.py
```

4. Выбрать желаемый `strings.xml` или `dialogsglobal.xml` файл.
5. Указать нужна ли сортировка по персонажам (актуально для `strings.xml`). 
6. Увидеть результат.

Результат сохраняется **в папку, где расположен исходный файл** в формате `[имя_файла]_dialogues.json`.


---
## Пример вывода:

### dialogsglobal.xml

```json
"r1m4_UnknownBar_Customer5_1_N01": [
    {
        "speaker": "Неил",
        "text": "Сижу, грущу…"
    },
    {
        "speaker": "ИГРОК",
        "text": "А что так?"
    },
    {
        "speaker": "Неил",
        "text": "А кому сейчас легко?"
    },
    {
        "speaker": "ИГРОК",
        "text": "Ну и сиди тут…"
    }
],
```
